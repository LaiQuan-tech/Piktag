1. 產品定位

PikTag 是一個基於 Link in Bio + Semantic Tag + 排序演算法 的人脈管理平台，結合輕量 CRM 功能，幫助使用者記住每一個重要的人。

1.1 核心價值

  • Link in Bio — 個人公開頁面（pikt.ag/username），展示社交連結與標籤
  • Semantic Tag — 語意標籤系統，自動歸類同義標籤（#媽祖 = #天上聖母 = #Mazu）
  • Smart Sorting — 標籤 6 級排序演算法（isPinned → isPicked → isHidden → pickCount → isMutual → position）
  • 輕量 CRM — 跟進提醒、生日提醒、人脈洞察

1.2 目標市場

全球多語系支援（15 種語言），主要市場：台灣、日本、韓國、東南亞、北美

3.2 好友頁（ConnectionsScreen）

  • Stories Bar — IG 風格橫向滾動，只顯示已追蹤好友的近況，漸層圈（藍→紫→粉），點選後標記已讀消失
  • 好友清單 — 排序（最新/最舊/字母/地點）、標籤篩選（前 10 個最多人有的）
  • 好友地圖 — 全螢幕 Google Maps 顯示好友位置
  • 朋友數 + 摯友數 — Header 顯示
  • 整理新朋友 — Banner 連結到 ActivityReview

3.3 搜尋頁（SearchScreen）

  • 三個 Tab — 熱門標籤 / 附近標籤 / 搜尋紀錄
  • 語意分類篩選 — 個性 / 職業 / 興趣（動態從 DB 取得）
  • 趨勢指標 — 近 7 天成長 ≥3 人或 ≥20% 顯示 TrendingUp
  • 多標籤交集搜尋 — 選多個標籤 → 按搜尋 → 找同時有這些標籤的人
  • 搜尋欄 — 支援 # 號去除、多分隔符（空格/逗號/頓號）、搜尋結果顯示語意類型
  • Smart 推薦 — 「你可能想認識」基於共同標籤推薦
  • 語系優先 — 依手機系統語言排序熱門標籤

3.4 QR Code Tab（AddTagScreen）

  • 日期選擇 — 快速日期（今天/明天/本週末/下週）+ 日曆
  • 地點選擇 — 全螢幕地圖 Modal（Google Places API + Autocomplete）
  • 標籤選擇 — 預設熱門標籤（6 個）+ 自訂標籤
  • QR Code 產生 — 含日期/地點/標籤資訊
  • 預設組合 — 儲存/載入/套用標籤組合

4.2 概念歸類系統（tag_concepts）

  • tag_concepts — 語意概念表，含 embedding vector(3072)
  • tag_aliases — 別名對應（多對一）
  • auto-link-concepts — Edge Function，用 Gemini Embedding 自動歸類
  • find_similar_concepts() — PostgreSQL 函式，cosine similarity ≥ 0.85
  • 範例：#媽祖 ≈ #天上聖母 ≈ #Mazu → 同一 concept
