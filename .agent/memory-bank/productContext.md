# Product Context: PikTag

## Overview
PikTag is a unified mission control and personal CRM system designed for high-efficiency collaboration and personal relationship management. It serves as a dashboard for tracking development progress, version sentinel for GitHub integration, and a mobile-first CRM for managing professional and personal connections.

## Problem Statement
Traditional project management tools are often detached from personal network management and real-time collaboration signals. Users need a system that bridges the gap between their development workflows (GitHub, AntiGravity) and their interpersonal networks (Contacts, Social Links, CRM).

## Target Users
- Software Developers and Architects.
- Product Managers.
- High-frequency networkers and professional collaborators.

## Key Features
- **Mission Control Dashboard**: Unified view of feature progress and version history.
- **Smart CRM**: Automatic reminders for birthdays, anniversaries, and contract expiries.
- **Biolink Tracking**: Social link management with real-time click notifications.
- **Hyper-local Networking**: GPS-based discovery of contacts and trending tags (e.g., "Who do I know in this location?").
- **Social Stats**: Comprehensive analytics on connection growth and interaction frequency.
- **Tagging System**: Deep categorization of contacts with trending analysis.

## Success Criteria
- Seamless synchronization between mobile and web platforms via Supabase.
- Reliable delivery of time-sensitive CRM notifications.
- High user engagement with social statistics and local networking features.
- Performance: Liquid Glass UI responsiveness across all supported devices.
